# 灵妆用户使用手册

## 安装灵妆

1. 打开 [LingGlow Releases](https://github.com/AI-Scarlett/LingGlow/releases/latest)。
2. 下载当前 Release 中的 macOS DMG，并同时下载 `SHA256SUMS`。
3. 打开 DMG，把“灵妆.app”拖入“应用程序”。
4. 首次启动时确认安装包来自本仓库的正式 Release，且 macOS 没有报告签名或公证错误。
5. 灵妆不显示在 Dock；点击 macOS 菜单栏的灵妆图标即可打开桌面窗口。

系统要求：macOS 13 或更高版本。仅从 `AI-Scarlett/LingGlow` 官方 Release 下载。

## 浏览和下载皮肤

1. 打开灵妆，进入“皮肤”。
2. 在“目标应用”选择 WorkBuddy、豆包或 Codex。
3. 灵妆会从 GitHub 读取皮肤名称、预览图、免费/VIP 标记、下载大小和适配范围。
4. 未安装的皮肤显示“下载”。点击后只下载这一套皮肤，不会重新下载或升级整个 App。
5. 下载完成后，免费或已购皮肤显示“应用”；未购买的 VIP 皮肤显示“试用一次”。
6. 皮肤有新版本时显示“更新”；点击即可替换本地皮肤，仍不需要更新灵妆 App。
7. 可用搜索框按名称、系列或标签搜索，也可按运动、幻想、自然、极简、艺术等分类筛选。
8. 页面提示目录有更新时点击“拉取新模板”；也可随时手动刷新，不必升级灵妆 App。
9. “推荐”展示当前主打主题；“我的皮肤”只展示已经下载或单独购买的皮肤。

远程皮肤使用两级校验：先核对整个皮肤包的 SHA-256，再核对定义文件和每张 WebP 的 SHA-256。校验失败时不会安装。

## 管理“我的皮肤”

- 每张卡片会标记适用的 Codex/GPT、WorkBuddy、豆包客户端。
- “已下载”表示完整皮肤包已经保存在本机。
- “已购买”表示授权已绑定，但皮肤包可能还没有下载。
- “有更新”表示本地包与远程目录版本不同，点击即可更新。
- 顶部更新数量按皮肤 ID 去重，同一套皮肤适配多个 Agent 时只计算一次。

## 切换中文或 English

1. 打开左侧“设置”。
2. 在“界面语言”选择“中文”或“English”。
3. 界面会立即切换，不需要重新启动灵妆。

灵妆只提供简体中文与英文两套界面语言。

## 自动更新灵妆

1. 打开左侧“设置”。
2. 开启“启动时自动检查并安装安全更新”。
3. 灵妆会读取官方 latest.json，自动选择 Apple Silicon 或 Intel 安装包。
4. 安装前会校验 SHA-256、Bundle ID、Developer Team ID、签名与 Apple 公证票据。
5. 也可以点击“立即检查”手动获取版本；应用更新与皮肤目录更新互不影响。

自动更新默认开启，可随时关闭。未通过任一安全校验的安装包不会执行。

## 安装皮肤制作 SKILL

1. 打开左侧“设置”。
2. 在“制作自己的皮肤”卡片点击“一键安装 SKILL”。
3. 重新打开 Codex。
4. 使用 \$skin-abstract 描述皮肤名称、明暗模式、配图和适配要求。

详细步骤、图片规格和提示词见 [皮肤制作手册](CREATE-SKIN-WITH-SKILL.md)。

## 应用皮肤

1. 保存目标 Agent 中尚未发送的文字和未完成任务。
2. 在已下载皮肤卡片点击“应用”。
3. 阅读重启确认，确认后灵妆只重启所选 Agent。
4. 如果皮肤声明为深色，请保持 Agent 自身为深色模式；浅色皮肤同理。不要在皮肤生效后单独切换 Agent 的明暗模式。
5. 需要撤销时点击“恢复原版”。恢复不会删除对话、项目、账号或 Agent 配置。

下载皮肤不等于应用皮肤。即使已下载，灵妆仍会在应用前检查 Agent 官方签名、版本、适配级别和当前授权。

## VIP 皮肤的一次性试用

- 每套未购买的 VIP 皮肤默认只能试用一次。
- 成功应用后，当前 Agent 进程可以继续使用该皮肤；下一次重启 Agent 后不再生效。
- 一次性试用成功启动后立即记为已使用，不能再次试用同一套皮肤。
- 输入有效 VIP 密钥，或兑换绑定该 `skinId` 的永久皮肤密钥后，可以重复应用。
- 7 天本机功能试用不会把所有 VIP 内置皮肤变成无限次使用。
- 仅下载、查看预览或取消重启确认不会消耗试用；只有 Agent 成功启动并应用后才消耗。

## 购买前同意协议

- 购买前必须阅读并同意《购买说明》和《隐私政策》。
- 未勾选协议时购买按钮仍可点击，但不会直接打开支付页。
- 灵妆会弹窗解释原因，并提供查看协议或“同意并继续购买”的入口。
- 数字授权和虚拟服务除适用法律或 Dodo Payments 规则另有规定外不支持退货退款。

## 离线和缓存

- 成功读取过目录后，灵妆会在本机缓存目录索引。
- 没有网络时，可以继续使用已经下载的皮肤。
- 首次启动且没有网络时，仍可使用内置的“极光青 Free”免费皮肤，以及“C罗灵感·葡萄牙7号星夜”“功夫女足·无厘头绿茵夜”两套 VIP 模板。VIP 仍遵守授权与一次性试用规则。
- 已下载皮肤位于 `~/Library/Application Support/Codex Skin Studio/remote-theme-packs/`。

## 常见问题

### 看得到皮肤但按钮显示“下载”

这是正常的远程目录状态。预览图很小，实际背景、Hero 和机器人图片只有在点击“下载”后才保存到本机。

### 下载后不能应用

检查目标 Agent 是否已安装、是否为受支持版本，以及 VIP/单皮肤授权是否有效。Codex 的 `generic-safe` 模式只开放背景、色板和基础玻璃，不会强行应用尚未验证的布局能力。

### GitHub 暂时无法访问

灵妆会读取上一次成功缓存的目录。已安装皮肤不受影响；新皮肤要等网络恢复后才能下载。

### 皮肤文字与背景不匹配

确认皮肤明暗模式与 Agent 自身明暗模式一致，然后重新应用。弹窗、菜单和输入框应使用语义色，而不是直接取背景图片颜色。

## 安全说明

远程皮肤只允许声明式 JSON 和静态 WebP，不允许 JavaScript、CSS、SVG、可执行文件、任意网络地址或源代码。完整协议见 [远程皮肤目录说明](REMOTE-SKIN-CATALOG.md)。
