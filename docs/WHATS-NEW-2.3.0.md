# 灵妆 2.3.0 更新说明

发布日期：2026-07-20  
官方网站：https://aiaiai.homes/

## 皮肤库

- 新增“推荐 / 全部 / 我的皮肤”三栏。
- “我的皮肤”只显示已下载或单独购买的皮肤。
- 每张个人皮肤卡显示适用 Agent、已下载、已购买和有更新状态。
- 更新数量按皮肤 ID 去重。
- 西班牙世界杯夺冠与八仙九套系列进入推荐栏，全部免费。

## 客户端

- 新增中文和 English 两种语言资源。
- 设置页可选择语言，重新启动灵妆后生效。
- WorkBuddy、豆包、Codex/GPT 筛选使用本机官方 App 图标。
- 菜单栏使用灵妆 App Icon 中的丝带主体。

## 购买体验

- 购买仍要求同意购买说明和隐私政策。
- 未勾选时购买按钮不再灰置；点击会弹窗说明原因。
- 弹窗可查看协议或选择“同意并继续购买”。

## 修复

- 修复八仙主题 `codex.banner.opacity` 超出允许范围。
- 修复世界杯主题透明度边界值。
- 同步 Theme Pack 定义 SHA-256 锁，避免 WorkBuddy 拒绝加载。
- 25 套 WorkBuddy/Codex 皮肤与 18 套豆包皮肤均完成服务层逐套解析。
