# 灵妆（LingGlow）隐私政策

生效日期：2026 年 7 月 19 日

## 1. 适用范围

本政策适用于灵妆 macOS 客户端及其公开的皮肤目录、升级服务和购买授权功能。

## 2. 灵妆在本机处理的数据

灵妆会在用户的 Mac 上保存皮肤配置、已下载的皮肤资源、排程设置、目标 Agent 的兼容性检测结果以及用户主动选择的图片。授权码、Dodo License 激活实例和本机绑定信息保存在 macOS Keychain 中，不写入公开日志，也不会嵌入安装包。

用户可通过灵妆移除本机授权缓存，也可卸载应用并删除灵妆的本地数据目录。

## 3. 网络请求

灵妆仅在完成相应功能时连接以下服务：

- GitHub：读取公开皮肤目录、下载用户选择的皮肤包、检查应用更新。
- Dodo Payments：打开官方托管支付页面，并调用公开的 License 激活、验证和停用接口。
- Apple：macOS 可能检查 Developer ID 签名、公证票据和系统安全状态。

灵妆不在客户端嵌入 Dodo 商户 API Key，不要求 Webhook，也不自行收集或保存银行卡号、支付账户密码、税务资料或完整支付凭据。

## 4. 支付数据

Dodo Payments 作为 Merchant of Record 独立处理结算、税费、发票、欺诈防控、退款和争议。用户在 Dodo 页面提交的信息受 [Dodo Payments 隐私政策](https://dodopayments.com/privacy-policy)约束。

## 5. 分析与广告

当前版本不包含第三方广告 SDK，不出售个人信息，也不上传用户在 WorkBuddy、豆包或 Codex 中的聊天内容。皮肤应用功能只在本机处理必要的目标应用界面配置。

## 6. 数据安全与保留

灵妆使用 macOS Keychain 保存授权材料；远程皮肤包与升级包在安装前校验完整性。授权记录会保留到用户停用设备或清除本机授权。已下载皮肤保留到用户删除或卸载相关数据。

## 7. 用户权利与联系

用户可以在灵妆中停用设备、清除授权缓存，并删除已下载皮肤。有关隐私、授权或数据删除的问题，可通过 [LingGlow GitHub Issues](https://github.com/AI-Scarlett/LingGlow/issues) 联系。

本政策可能因功能、法律或第三方服务变化而更新；更新版本会在 GitHub 仓库公布。
