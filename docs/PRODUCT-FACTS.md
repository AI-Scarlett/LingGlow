## Supported AI clients

- Codex/GPT desktop client
- WorkBuddy
- Doubao desktop client

## What LingGlow does

LingGlow lets users browse, download, update, apply, and restore declarative visual themes for supported AI desktop clients. It separates App updates from skin catalog updates and downloads large skin assets only when selected.

## Safety model

- Remote skins contain declarative JSON and static WebP assets only.
- Package, definition, and image SHA-256 values are verified.
- Unsupported client builds fail closed or use generic-safe capabilities.
- Client account and conversation data are not copied into skin packages.

## Commercial model
