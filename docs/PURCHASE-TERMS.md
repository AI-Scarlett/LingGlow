# 灵妆（LingGlow）购买说明

生效日期：2026 年 7 月 19 日

## 1. 商品性质

灵妆销售的是数字授权、订阅权益和虚拟皮肤使用权，不寄送实物。付款成功后，Dodo Payments 会通过支付完成页、邮件或 Customer Portal 交付 License Key，用户需在灵妆“授权”页面粘贴并激活。

**数字授权一经交付或激活，除适用法律、银行卡组织规则或 Dodo Payments 退款政策另有强制规定外，不支持退货退款。** 请在付款前确认商品类型、订阅周期、价格、币种、适用税费和目标 Agent 的兼容性。

## 2. 正式商品

- VIP 月度订阅：订阅有效期间使用全部 VIP 皮肤、自定义皮肤和排程功能。
- VIP 年度订阅：按年自动续订，权益范围与月度订阅一致。
- 单套皮肤永久授权：首次兑换时在本机选择一套 VIP 皮肤并绑定。
- 自定义皮肤位永久授权：解锁一个由 License Key 确定的本机自定义位。

支付页面显示的实时价格、币种、税费、折扣和续订日期为最终结算依据。

## 3. 订阅与取消

月度和年度 VIP 属于自动续订订阅。用户应在下一计费日前通过 Dodo Payments 的 Customer Portal 管理或取消订阅。取消通常阻止下一次续费，不等同于退还已经完成的当前周期费用；具体以 Dodo 页面、适用法律和支付平台规则为准。

## 4. 退款、拒付与撤销

灵妆是即时交付的虚拟商品，原则上不接受因不喜欢皮肤、未阅读说明、目标 Agent 后续改版或用户自行更改系统环境提出的退货退款。

如果适用法律或 Dodo Payments 判定必须退款，或交易发生拒付、欺诈、订阅取消/到期，Dodo 可以撤销 License。灵妆会在启动、手动刷新或定期联网验证后停止相应权益；未配置 Webhook 的无后端版本不会承诺秒级撤销。

## 5. 试用与兼容性

未购买的 VIP 皮肤每套仅可试用一次；成功启动目标 Agent 后即记为已试用，下次重启 Agent 后失效，不能再次试用。试用不构成购买，也不延长订阅。

WorkBuddy、豆包和 Codex 均为独立第三方产品，其升级可能影响皮肤兼容性。灵妆会提供兼容性检测、恢复原版和皮肤更新，但不承诺第三方应用所有未来版本永久保持同一界面。

## 6. License 使用规则

License Key 仅供购买者按商品说明激活使用，不得公开分享、转售、绕过设备限制或用于未授权分发。授权码保存在 macOS Keychain。停用设备会调用 Dodo 公共 License API 释放当前激活实例。

单套皮肤和自定义位的选择在无后端版本中保存在当前 Mac 的 Keychain；它不是跨设备中央账户绑定。清除本机数据可能需要重新使用原 License Key 激活，且仍受 Dodo 的激活数量和有效状态限制。

## 7. 支付与隐私

Dodo Payments 作为 Merchant of Record 处理支付、税费、发票、退款和争议。灵妆不保存银行卡信息，也不会把商户 API Key 放进客户端。详情请查看[灵妆隐私政策](PRIVACY.md)与 [Dodo Payments 隐私政策](https://dodopayments.com/privacy-policy)。

## 8. 联系

购买前如有兼容性或授权疑问，请通过 [LingGlow GitHub Issues](https://github.com/AI-Scarlett/LingGlow/issues) 联系。涉及支付状态、发票、退款或银行卡争议时，也可能需要联系 Dodo Payments 支持。
